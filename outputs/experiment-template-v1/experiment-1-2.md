## 实验1: 策略差异验证

### 目的
验证 A/B/R 三种定价策略在 Shi-Graph 中的表现差异，对比 ABM 与 PDE 模型的区分度。

### 设置
| 参数 | 值 |
| --- | --- |
| 节点数 | 300 |
| 步数 | 1000 |
| A组 | 自由定价 |
| B组 | 参考价策略 |
| R组 | 随机定价（对照） |
| 噪声 | 5%（实验2中可调: 1%/5%/10%/20%） |
| 衰减率 γ | 0.20 |
| 使用事件 ψ | 0.15（代码中 SYPHON_PSI=0.15，同一参数） |
| phi权重 | 0.60（增强版锁定值） |

### 观测指标
1. **Gini_Phi** — Phi分布不平等程度
2. **Gini_P** — 价格分布不平等程度
3. **存活率** — A/B/R 各组存活节点数
4. **Mean Phi** — 平均 Phi 水平
5. **A/B 区分度** — A组 vs B组的指标差异倍数
6. ~~新指标~~ ✅ **内容多样性指数** — Shannon 信息熵 H = -Σ p_i log p_i，衡量内容分布多样性

### 历史数据参考

#### 猫先森 ABM（300节点，1000步）
| 组 | 存活 | Gini_P | Gini_Phi | Step 100 Gini_P |
| --- | --- | --- | --- | --- |
| A | 63 | 0.2848 | 0.70 | 0.071 |
| B | 53 | 0.2478 | 0.73 | 0.038 |
| R | 44 | 0.2139 | 0.67 | 0.301 |

#### 云子 PDE Baseline（300节点，1000步）
| 组 | 存活 | Gini_P | Gini_Phi | Step 100 Gini_P |
| --- | --- | --- | --- | --- |
| A | 300 (100%) | 0.008 | 0.000 | 0.071 |
| B | 300 (100%) | 0.038 | 0.000 | 0.038 |
| R | 300 (100%) | 0.314 | 0.000 | 0.301 |

### 关键发现
- ABM 中 A/B 区分度 ~2.4x
- PDE Baseline 中 A/B 区分度 ~1.87x（增强版 6.40x 但定价差异被抹平）
- 两种物理对策略差异的敏感度不同

## 实验2: 抗博弈测试

### 目的
验证透明推荐算法在不同 noise_level 下的抗博弈能力，评估"白盒+可控噪声" vs "黑盒"的区别。

### 设置
| 参数 | 值 |
| --- | --- |
| 节点数 | 300 |
| 步数 | 1000 |
| **noise_level (ε)** | **可调: 0% / 5% / 10% / 20%** |
| 其他参数 | 同实验1 |
| phi权重 | 0.60（增强版锁定值） |

> 注: 实验1默认噪声5%作为基线；实验2中 noise_level 扫描为独立变量，含1%档位。

### 核心变量
- **noise_level (ε)**: 推荐算法中的随机扰动比例
  - ε=0%: 完全透明，无噪声，可能被完全博弈
  - ε=1%: 轻度噪声（增强版锁定值）
  - ε=5%: 默认值，中度透明噪声
  - ε=10% / 20%: 高不确定性

### 假设
- H2a: ε=0% 时，A+B 方案 Agent 能通过精确博弈获得显著优势
- H2b: ε 增加时，博弈优势减弱，系统趋向更公平的分布
- H2c: ε 过高（>10%）时，推荐算法的有效性下降

### 创作者行为：民间理论层

> 哪吒拍板: A+B 方案 Agent 应模拟"猜测算法"行为，而非完全随机

#### 行为模型
| 方案 | 行为描述 | 复杂度 |
| --- | --- | --- |
| A | 观测近期使用模式 → 推断推荐偏好 → 调整定价 | 高（"猜测算法"） |
| B | 基于参考价，微调但不过度博弈 | 中 |
| C (可选) | 完全随机，无策略 | 低（对照） |

#### 民间理论定义
- Agent 不完全了解推荐算法，但能观测部分输出
- 通过有限样本"猜测"算法的行为模式
- 这模拟了真实创作者在"部分透明"环境下的行为

### 透明度原则
- **噪声参数 ε 是公开的**（白盒参数，非黑盒）
- 创作者可以计算期望收益
- 这与 Web2 黑盒推荐有本质区别

### 观测指标
同实验1，新增：
- **博弈收益比** — A/B 组在高博弈 vs 低博弈场景的收益差异
- **ε敏感性曲线** — Gini_Phi vs ε 的关系
