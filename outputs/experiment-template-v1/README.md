## 概述

本模板包含两个实验设计文档：
- **实验1 + 2**：策略差异验证 + 抗博弈测试
- **实验3 + 5**：透明度影响 + 长期演化

每条建议在群聊中经哪吒拍板，均已纳入设计。

## 三条建议（哪吒拍板）

1. **内容多样性指数** — 作为观测指标加入实验1和2
2. **创作者"民间理论"行为** — A+B 方案 Agent 模拟猜测算法，非完全随机
3. **随机性抗博弈** — noise_level (ε) 作为实验2可调参数，默认5%，扫描0%/5%/10%/20%
