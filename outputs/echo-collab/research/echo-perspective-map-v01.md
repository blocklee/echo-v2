## 0. 为什么需要视角地图

ECHO 由多个独立视角构成，每个视角从自己的出发点回答同一个问题："Φ 该怎么演化？"。

视角地图不是汇总——是**边界定义**：明确每个视角的责任范围、输入输出、以及**不触碰**的区域。

## 1. 四个核心视角

### 1.1 物理层 · 场论视角（云子/哪吒）

| 维度 | 内容 |
|------|------|
| **文档** | `docs/architecture/field-theory-v0.5.md` |
| **核心概念** | Φ 是归一化势位场；离散区块是有损压缩；连续链存储演化规则而非快照 |
| **关键定理** | 定理 1.1（离散链信息下界）、定理 1.2（连续链信息效率） |
| **输出** | 设计哲学、离散→连续映射、六档最优性（β*=6.2） |
| **责任范围** | 定义 Φ 的物理意义、为什么区块链需要连续链 |
| **不触碰** | 不推导 PDE 稳定性（数学层）、不做 witness 压缩（信息论层） |

**边界**：场论回答"Φ 是什么"，不回答"Φ 会不会爆炸"。

---

### 1.2 数学层 · PDE 有界振荡视角（雨娃/云子/哪吒）

| 维度 | 内容 |
|------|------|
| **文档** | `research/pde-bounded-oscillation-v03.md` |
| **核心概念** | 系统健康 ≠ 收敛到不动点；系统健康 = Φ(t) 在 V* 内有界振荡 |
| **关键方程** | ∂Φ/∂t = D·L·Φ − γ·Φ + S·[1+k·tanh(Φ/Φ₀)] − β·Φ³ |
| **关键阈值** | V* = [S·(1+k)·N√N/β]^(2/3)（v0.3.2 修正后） |
| **P0 实验** | γ_min ≈ 0.003（猫先森 commit ffbd4564） |
| **输出** | 有界性证明（β>0 保证）+ 稳定性证明（γ>0 保证） |
| **责任范围** | 定义系统的动力学方程、证明有界性+稳定性 |
| **不触碰** | 不做合约接口设计（合约层）、不做信息论 bound（信息论层） |

**边界**：PDE 回答"Φ 怎么演化"，不回答"Φ 在链上怎么验证"。

**关键发现（v0.3.2 新增）**：
- γ 项符号：−2γ·V（能量流出，不是流入）
- γ=0 时 V_steady = V*（边界平衡，非内部稳定）
- γ>0 从"工程建议"升级为"安全必要条件"
- 围栏比喻：β=围栏高度，γ=恢复力坡度

---

### 1.3 信息论层 · Witness 预算视角（Seaman_bot）

| 维度 | 内容 |
|------|------|
| **文档** | `research/zk-witness-budget.md` |
| **核心概念** | 链下预计算 V witness，链上只做比较 |
| **关键 bound** | β^(2/3) scaling：β 翻倍 → witness 减少 37%（不是 50%） |
| **关键参数** | λ：Layer4 目标函数权重，由 Pareto 分析 + 决策者偏好决定 |
| **P0 数据** | λ Pareto frontier：不同 λ 下的 Gini vs 证明成本 trade-off |
| **输出** | witness 压缩 bound、λ trade-off 曲线 |
| **责任范围** | 证明链下预计算的信息论上限、定义证明成本 |
| **不触碰** | 不做合约实现（合约层）、不做 PDE 稳定性（数学层） |

**边界**：信息论回答"Φ 的证明成本"，不回答"Φ 本身对不对"。

---

### 1.4 跨层视角 · 一致性审视角（Talus）

| 维度 | 内容 |
|------|------|
| **文档** | `research/pde-bounded-oscillation-v03-cross-layer-addendum.md` |
| **核心概念** | 三层（物理/数学/信息论）之间的一致性、合约映射 |
| **三层映射** | 物理:V=ΣΦ² → 数学:dV/dt bound → 信息论:witness 证明 |
| **Gas 分析** | 链上 O(N) vs 链下 O(1)；动态调度阈值 N·gas_op > block_gas_limit/10 |
| **接口定义** | EnergyBound struct、BoundViolation 三档状态机、EIP-712 签名 |
| **信任模型** | 质押+签名 slash 机制 |
| **责任范围** | 定义三层之间的接口、Gas 成本、信任模型 |
| **不触碰** | 不推导数学证明（数学层）、不做信息论 bound（信息论层） |

**边界**：跨层视角不生产理论——它只确保三层理论的接口一致。

## 2. 层间接口索引

```
┌───────────────┐        ┌───────────────┐        ┌───────────────┐
│   物理层      │──V=ΣΦ²──│   数学层      │──dV/dt──│   信息论层    │
│  field-theory │        │ pde-bounded-  │        │ zk-witness-   │
│  -v0.5.md     │        │ oscillation   │        │ budget.md     │
└───────┬───────┘        └───────┬───────┘        └───────┬───────┘
        │                        │                        │
        │  Gas cost              │  λ 权重                │  witness bound
        │  10K (on-chain)       │  Pareto frontier       │  β^(2/3) scaling
        │  ↓                     │  ↓                     │  ↓
        │  跨层一致性审           │  跨层一致性审           │  跨层一致性审
        │  (Talus)               │  (Talus)               │  (Talus)
        └────────────────────────┼────────────────────────┘
                                 │
                          合约层 (Solidity)
```

### 2.1 物理→数学 接口

| 字段 | 物理层输出 | 数学层输入 | 一致性校验 |
|------|----------|----------|----------|
| Φ 物理单位 | 归一化势位 [1]（无量纲） | Φ ∈ [0,1] 无量纲 | ✅ 一致 |
| V 定义 | 未定义 | V = ΣΦᵢ² (L²) | ⚠️ 场论未定义 V，数学层自主定义 |
| 扩散项 | D·L·Φ（连续链自然项） | 2D·ΣΦᵢ·L·Φᵢ ≤ 0 | ✅ 一致 |

### 2.2 数学→信息论 接口

| 字段 | 数学层输出 | 信息论层输入 | 一致性校验 |
|------|----------|----------|----------|
| V* 公式 | [S·(1+k)·N√N/β]^(2/3) | 同上 | ✅ 一致 |
| γ_min | 0.003（P0 实验） | γ > 0 必要 | ✅ 一致（0.003 > 0）|
| dV/dt bound | ≤ 2S·(1+k)·√N·√V − 2β·V²/N | witness 构造依据 | ✅ 一致 |
| λ 权重 | 未涉及 | Pareto trade-off | ⚠️ 数学层无 λ，信息论层独立推导 |

### 2.3 跨层→合约 接口

| 字段 | 跨层定义 | 合约实现 | 一致性校验 |
|------|--------|--------|----------|
| EnergyBound struct | V_max, V_witness, deadline | Q32.32 定点数 | ⚠️ 待合约实现验证 |
| BoundViolation | NONE / SOFT_REVERT / HARD_REVERT | 状态机 | ⚠️ 待合约实现验证 |
| Gas 成本 | 链上 ~10K / 链下 ~5K | 实测 gas | ⚠️ P0 验证项 |
| γ_max | 0.4（占位）| uint256 gamma_max_scaled | ✅ 一致 |

## 5. 视角关系原则

1. **每个视角独立推导**：不依赖其他视角的结论，独立验证
2. **接口处交叉校验**：物理→数学→信息论→跨层，每层输出都是下层的输入
3. **冲突时以实验为准**：理论推导 → P0 实验 → 拍板
4. **不越界**：每个视角只做自己层的事，接口处由跨层审对齐
5. **文档边界清晰**：`field-theory-v0.5.md` 不含 PDE 推导；`pde-bounded-oscillation-v03.md` 不含 witness 压缩；`zk-witness-budget.md` 不含 Gas 成本
