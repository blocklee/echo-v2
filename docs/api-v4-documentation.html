响应字段
  • run_id — 唯一实验 ID
  • summary — 汇总指标（总事件/上链事件/总收入/满意度/生态健康度）
  • timeseries — 每天时序数据（活跃用户数/事件数/收入/新增资产/争议/满意度）
  • creator_results — 每个创作者的完整结果（收入/满意度/资产数/引用/改编/势位变化/配置历史）
  • asset_lifecycle — 每个资产的完整轨迹（创建日/势位时间线/总收入/最终状态）
  • user_states — 每个用户最终状态（角色/情绪/钱包/交互次数/人格）
  • s_graph_data — S-Graph 图结构（节点/边/张量/网络统计）
  • snapshots — 每日快照（完整生态状态）
  • events — 所有事件日志
POST/run/tournament统计检验

配置策略锦标赛，对比多种策略，含 t-test/p值/置信区间/Cohen's d

请求体

  "simulation_config": {"duration_days": 30, "num_creators": 6, "num_users": 20},

🔬 统计检验说明

Welch's t-test（独立样本，不等方差）
Cohen's d 效应量解释：
  • < 0.2 — 可忽略
  • 0.2 ~ 0.5 — 小效应
  • 0.5 ~ 0.8 — 中效应
  • > 0.8 — 大效应
95% 置信区间基于 t 分布计算
