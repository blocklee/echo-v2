草案 v1.0 📅 2026-05-14 👤 雨娃 · 模拟机制设计小组 📋 待团队确认

文档目标：明确前端、后端、模拟引擎之间的数据契约，支撑团队并行开发。

本文档基于现有 echo_simulator_api.py 和 visualization_mesa.html 的已实现接口整理，覆盖 5 个 API 端点、完整数据类型定义、可视化数据格式及错误处理规范。

2 API 端点定义

GET /api/v1/health

健康检查 · 返回服务状态和能力列表

{ "status": "ok", "version": "1.0.0", "engine": "mesa", "capabilities": ["single_run", "tournament", "ab_test", "robustness"] }
GET /api/v1/config

获取默认配置 · 返回模拟器默认参数和可选枚举值

{ "default_simulation": { "duration_days": 30, "num_creators": 6, "num_users": 20, "seed": null }, "creator_types": ["保守型", "开放型", "平衡型", "策略型"], "user_roles": ["浏览者", "引用者", "改编者", "传播者", "付费者", "争议者"], "four_powers": { "铸": "铸造权", "衍": "衍生权", "扩": "扩展权", "收": "收益权" } }
POST /api/v1/run

单次模拟运行 · 完整的创作者/用户/环境配置

请求体含完整配置，响应含时序数据、创作者结果、资产生命周期、S-Graph数据

// 请求：完整模拟配置 { "simulation_config": { "duration_days": 30, "num_creators": 6, "num_users": 20, "seed": 42 }, "creators": [ { "id": "creator_1", "name": "林水墨", "type": "保守型", "asset_config": { "铸": { "enabled": true, "price": 100 }, "衍": { "enabled": true, "max_derivatives": 3, "fee_percent": 15 }, "扩": { "enabled": true, "platforms": ["web", "app"] }, "收": { "enabled": true, "revenue_share": 70 } }, "initial_work": { "title": "《晨曦》", "category": "视觉艺术", "quality_score": 8.5 } } ], "users": [...], "environment_params": { "platform_fee_percent": 5, "discovery_algorithm": "trending", "market_growth_rate": 0.02 } }
// 响应：模拟结果 { "run_id": "run_20260514_001", "status": "completed", "summary": { "total_events": 1196, "onchain_events": 310, "total_revenue": 2847.50, "creator_satisfaction": 6.8, "ecosystem_health": 0.72 }, "timeseries": [ { "day": 1, "active_users": 18, "events": 45, "revenue": 12.00 } ], "creator_results": [...], "asset_lifecycle": [ { "asset_id": "asset_1", "shi_timeline": [ { "day": 1, "position": [0, 0, 0] }, { "day": 30, "position": [2, 3, 2] } ] } ], "s_graph_data": { "nodes": [...], "edges": [...] } }

3 核心数据类型定义

3.1 创作者配置 CreatorConfig

{ "id": "creator_1", // 唯一标识 "name": "林水墨", // 创作者名称 "type": "保守型", // 枚举：保守型/开放型/平衡型/策略型 "asset_config": { "铸": { "enabled": true, "price": 100, "currency": "ECHO" }, "衍": { "enabled": true, "max_derivatives": 3, "fee_percent": 15, "approval_required": false }, "扩": { "enabled": true, "platforms": ["web", "app"], "auto_approve": true }, "收": { "enabled": true, "revenue_share": 70, "instant_payout": true } }, "initial_work": { "title": "《晨曦》", "category": "视觉艺术", "quality_score": 8.5 // 0-10 }, "behavior_params": { // 可选：行为参数 "adaptability": 0.5, // 配置调整意愿 0-1 "quality_focus": 0.8, // 质量vs数量偏好 "social_activity": 0.6 // 社区参与度 } }

3.2 用户行为配置 UserConfig

{ "id": "user_1", "role": "浏览者", // 枚举：6种角色 "behavior_profile": { "activity_level": 0.7, // 每日活跃概率 "preference_categories": ["视觉艺术", "音乐"], "price_sensitivity": 0.3, // 价格敏感程度 "discovery_bias": "trending", // 发现偏好：trending/random/category/creator "interaction_depth": "engaged" // 互动深度：surface/deep/engaged }, "memory_params": { "short_term_window": 7, // 短期记忆窗口（天） "trend_sensitivity": 0.5 // 趋势敏感度 } }

3.3 势位评估结果 ShiPosition

{ "time_dimension": 2, // 时间维：0-3，基于事件数量 "space_dimension": 3, // 空间维：0-3，基于平台覆盖 "relation_dimension": 2, // 关系维：0-3，基于关系事件数 "overall_level": 2, // 综合势位：0-3 "timestamp": "2026-05-14T09:17:00Z" }

4 错误处理规范

标准错误格式：

{ "error": { "code": "INVALID_CONFIG", "message": "创作者配置中铸权力价格不能为负数", "field": "creators[0].asset_config.铸.price", "suggestion": "请设置大于等于0的价格，或使用-1表示免费" } }
错误码说明HTTP
INVALID_CONFIG配置参数错误400
SIMULATION_FAILED模拟运行失败500
TIMEOUT模拟超时（大规模运行）504
UNSUPPORTED_MODE不支持的实验模式400
RATE_LIMIT请求频率限制429
