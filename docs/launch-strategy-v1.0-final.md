## 三、传播核心话术（合规终稿版）

### 禁用词（严格规避）
- ❌ 「投资」「回报」「增值」「高收益」「稳赚」「天天分红」
- ❌ 「投注」「下注」「赌博」「博彩」
- ❌ 「证券」「代币增值」「内部认购」

### 可用话术
| 序号 | 话术 | 用途 |
|---|---|---|
| 1 | 「铸一张卡 = 开一个被动收入账户」 | 铸造入口 |
| 2 | 「被编入卡组 = 躺着收 45% 创作收益」 | 编排激励 |
| 3 | 「编排者组出强卡组 = 每次对战抽 25%」 | 组队激励 |
| 4 | 「势位越高，分账乘数越大（最高 2.25 倍）」 | 成长激励 |
| 5 | 「创作者真正拥有作品版权，ECHO 链上存证作为辅助证据」 | IP 保护 |
| 6 | 「对战消耗/质押机制」替代「对战投注」 | 机制说明 |

### 经济模型最终参数（来自猫先森 v0.1/v0.2）
| 参数 | 数值 | 说明 |
|---|---|---|
| **所有权（Full）** | Premium 45 MEER/月，Basic 30 MEER/月 | 四权定价 |
| **使用权（Usage）** | 10 MEER/次（基础），2 MEER/次（经验价） | 首月减免 |
| **衍生权（Derivative）** | 250 MEER/季（基础） | 另首月减半 |
| **扩展权（Extension）** | 高级 40%，普通 20% | 四权定价 |
| **势位收益乘数** | ✦=0.1 / ✦✦=0.3 / ✦✦✦=0.6 / ✦✦✦✦=1.0 / ✦✦✦✦✦=1.5 | 5档体系 |
| **对战乘数** | 战胜 ×1.2，战败 ×0.8 | |
| **衰减率** | 2%/天 | 30 天无编排触发 |
| **分账结构** | 公池30% + 三权30%（基于影响）+ 运营20% + 预留10% | BattleRevenue v1.1.1 |

### 风险提示（每个物料底部必须加）
> ⚠️ 《势位之战》为基于 ECHO 协议的使用权经济游戏。MEER 仅作为游戏内道具，不承诺任何投资回报。请理性参与，18+ 且实名认证。

## 四、合规重点（运营侧红线）

| 红线 | 说明 | 违规后果 |
|---|---|---|
| 代币定位 | MEER = 游戏道具/使用权，绝不涉及投资增值承诺 | 法律+监管风险 |
| 对战机制 | 消耗/质押，禁用「投注」「赌博」「下注」 | 博彩红线 |
| 法币禁区 | MVP 仅接受 MEER，暂不涉及法币充值提现 | AML/KYC 合规 |
| 年龄限制 | 18+ 实名认证，未成年人完全禁止 | 未成年人保护 |
| IP 归属 | 传播「创作者真正拥有作品」，非「平台拥有版权」 | IP 合规 |
| 审计背书 | SlowMist/CertiK 审计后作为信任背书 | 安全背书 |
| 争议解决 | CIETAC 仲裁，适用中华人民共和国法律 | 法律适用 |

## 六、合规清单关键引用（8 条修改意见整合）

1. **对战消耗/质押** → 原「对战投注」，已修正，传播禁用「投注」相关表述 ✅
2. **收益分配定性** → 「游戏内奖励/积分分配」，非「投资回报」 ✅
3. **法币充值删除** → MVP 阶段仅 MEER，无法币充值提现 ✅
4. **年龄限制** → 18+ 实名认证，未成年人完全禁止 ✅
5. **争议解决** → CIETAC 仲裁，适用中华人民共和国法律 ✅
6. **合约审计** → SlowMist 或 CertiK，建议 2-3 周内完成 ✅
7. **IP 归属** → ECHO 链上存证作为版权归属辅助证据 ✅
8. **运营阶段** → Phase 2 合规预留，法币充值需申请支付牌照 ✅
