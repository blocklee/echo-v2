# PDE 层：耦合形式 (min vs tanh) 稳定性对比实验设计

## 1. 实验目的
验证两种不同的耦合函数形式在势位演化中的稳定性、收敛速度以及物理直觉的符合程度。

## 2. 对比形式

### 形式 A：$\min(\Phi_g, \Phi_e)$ (最小项耦合)
**公式**：$\Phi_{total} = \Phi_g + \Phi_e + k \cdot \min(\Phi_g, \Phi_e)$
**特点**：
- **木桶效应**：耦合强度完全受限于较低的一方。
- **临界点**：在 $\Phi_g = \Phi_e$ 时存在不可导点。
- **物理意义**：强制要求双方势位平衡，一方崩塌则耦合项迅速消失。

### 形式 B：$\tanh(\Phi_g, \Phi_e)$ (平滑饱和耦合)
**公式**：$\Phi_{total} = \Phi_g + \Phi_e + k \cdot \tanh(\frac{\Phi_g + \Phi_e}{2}) \cdot \min(\Phi_g, \Phi_e)$
*(注：具体形式可调整为 $k \cdot \text{softplus}(\Phi_g, \Phi_e)$ 或其他平滑函数)*
**特点**：
- **平滑过渡**：消除 $\min()$ 的不可导点。
- **饱和效应**：在势位较高时，耦合增益趋向于常数，避免过快增长。
- **物理意义**：模拟一个具有最大耦合能力的饱和机制。

## 4. 评估指标

1. **$\Phi_{max}$ (峰值势位)**：
   - 评估是否会出现势位爆炸。
   - $\min()$ 与 $\tanh()$ 谁能更好地抑制峰值。

2. **$Gini$ 系数 (分布不均匀度)**：
   - 观察两种形式对势位分布的影响。
   - 是否导致严重的“贫富差距” (少数节点占据绝大多数势位)。

3. **收敛步数 (Convergence Step)**：
   - 到达稳态 $\Phi^* \pm \epsilon$ 所需的时间步数。
   - 验证 $\tanh()$ 是否能加快收敛速度。

4. **稳定性 (Stability)**：
   - 在不同 $k$ 值下，势位是否能保持震荡在一定范围内而非发散。

## 5. 实施计划

1. **代码实现**：在 `PDECubicBetaSimulator` 中实现 `coupling_type` 切换逻辑。
2. **运行实验**：执行 E0-E6 矩阵，记录每个实验的 $\Phi$ 时间序列。
3. **数据分析**：对比 $\min()$ vs $\tanh()$ 的 $\Phi_{max}$ 和 $Gini$ 曲线。
4. **报告输出**：给出 PDE 层的最终建议（推荐形式）。
