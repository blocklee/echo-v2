## 当前 PDE 方程

```
∂Φ/∂t = D·L·Φ − γ·Φ + S·[1 + k·tanh(Φ/Φ₀)] − β·Φ³
```

已知参数边界：

| 参数 | 范围 | 来源 |
|------|------|------|
| γ | 六档对应 {0.20, 0.15, 0.10, 0.05, 0.02, 0.00} | 六档系统 |
| D | [0, 1]，由用权档位决定 | 四权·用权 |
| β | 推荐 0.01；≥0.05 Gini 崩溃 | 实验结论 |
| k | 0.05（标准值）| 实验结论 |

## 有界振荡框架下的边界条件

在有界振荡框架（替代定点收敛）下，边界条件的角色发生变化：

### 1. 初始条件

```
Φ(t=0) = Φ_snapshot
```

Φ_snapshot = 上一次博弈均衡的冻结快照（雨娃确认，迭代求解）

### 2. 参数边界（六档）

| 档位 | γ | D | 作用 |
|------|---|---|------|
| 0 初九 | 0.20 | ~0 | 最大衰减，封闭生态 |
| 1 九二 | 0.15 | 小 | 较快衰减 |
| 2 九三 | 0.10 | 中 | 中等衰减 |
| 3 九四 | 0.05 | 大 | 较慢衰减 |
| 4 九五 | 0.02 | 很大 | 缓慢衰减 |
| 5 上九 | 0.00 | 1.0 | 无衰减，永久势位 |

### 3. β 振幅约束边界

在有界振荡框架下，β·Φ³ 项作为振幅约束：

```
|Φ(t) - μ(t)| ≤ δ  （系统健康条件）
```

其中 β 调控振荡幅度上限，而非推动收敛到定点。

### 4. γ → ∞ 截断问题

**问题**：imbalance_ratio → ∞ → γ → ∞（数学上无法处理）

**建议**：引入截断规则：

```
γ_i = min(gamma_base × (1 + α × imbalance_ratio_i), γ_max)
```

γ_max 待标定，建议从模拟数据中确定自然出现的最大值。
