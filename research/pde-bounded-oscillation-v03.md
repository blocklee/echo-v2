## 核心转向：从定点收敛 → 有界振荡

**原框架**（v0.4）：
- 系统健康 = PDE 收敛到稳定不动点 Φ*
- β 的作用：非线性饱和

**新框架**（有界振荡）：
- 系统健康 = Φ(t) 在有界区域内振荡，不发散
- β 的作用：约束振荡幅度

---

## ⚠️ v0.2 的根本性错误（云子指出）

v0.2 使用 V = max|Φᵢ − μᵢ| 作为 Lyapunov 函数，存在三个阻塞性问题：

1. **不可微**：max 范数在切换点不光滑，不是 C¹，无法对 PDE 使用链式法则
2. **量纲灾难**：dV/dt ~ [Φ]/[T]，β·Φ²·V ~ [Φ]³，两者量纲不匹配，除非 β 自带 [Φ]⁻²[T]⁻¹
3. **γ_max = 2× 是拍脑袋**：没有分岔理论支撑

**v0.3 声明**：v0.2 的 β 推导是数值观察，不是解析证明。本文档尝试严格化。

---

## 标准路径：L² 能量估计

### 选择 L² 而不是 max 范数的原因

- L² 范数是 C¹ 光滑的，可直接对 PDE 使用链式法则
- 反应-扩散方程的标准稳定性分析走 L² 或 H¹
- max 范数只在比较原理中使用，不适合做 Lyapunov 函数

### L² 能量函数定义

```
V(t) = ‖Φ(t)‖² = ∫|Φ(x,t)|² dx

或等价地（离散形式）：
V = Σᵢ Φᵢ²
```

**V 是 C¹ 光滑的** ✓

---

## PDE 方程

```
∂Φ/∂t = D·L·Φ − γ·Φ + S·[1 + k·tanh(Φ/Φ₀)] − β·Φ³
```

**注意**：Φ 是实数标量场（不是张量），这是量纲分析的前提。

---

## dV/dt 推导（严格）

```
V = Σᵢ Φᵢ²

dV/dt = Σᵢ 2Φᵢ · dΦᵢ/dt
       = Σᵢ 2Φᵢ · (D·L·Φᵢ − γ·Φᵢ + S·[1+k·tanh(Φᵢ/Φ₀)] − β·Φᵢ³)
       = 2D·Σᵢ Φᵢ·L·Φᵢ − 2γ·Σᵢ Φᵢ² + 2S·Σᵢ Φᵢ·[1+k·tanh(Φᵢ/Φ₀)] − 2β·Σᵢ Φᵢ⁴
```

**β 项的贡献**：
```
−2β·Σᵢ Φᵢ⁴ ≤ −2β·V² / N
```

其中用了 |Σᵢ Φᵢ⁴| ≤ (Σᵢ Φᵢ²)² / N = V⁴/N（柯西不等式）。

**阻尼条件**：
```
dV/dt ≤ −2β·V²/N + |其他项|
```

当 V > √(|其他项|·N/2β) 时，dV/dt < 0。

---

## ⚠️ 量纲问题（仍需明确）

| 项 | 量纲 |
|----|------|
| dV/dt | [Φ²]/[T] = [Φ²][T]⁻¹ |
| β·V²/N | β·[Φ]⁴/[N] |
| 其他项 | 待明确 |

**问题**：如果 Φ 是无量纲标量，则 dV/dt ~ [Φ²]/[T]，β·V²/N ~ β·[Φ⁴]。

两边仍可能不一致，取决于 Φ 是否真正无量纲。

**需要确认**：ECHO 中 Φ 的物理单位是什么？如果 Φ 无量纲，则 β 必须有 [T]⁻¹ 的维度才能使方程成立。

## γ_max 问题（重写）

**v0.2 的错误**：γ_max = 0.4 没有理论依据，是拍脑袋。

**正确的来源应该是**：从线性稳定性分析出发。

### 线性化分析

在 Φ ≈ 0 附近线性化（忽略高阶项）：
```
∂Φ/∂t ≈ D·L·Φ − γ·Φ + S·k·(Φ/Φ₀) + S

忽略空间项 D·L·Φ（假设均匀）：
∂Φ/∂t ≈ −(γ − S·k/Φ₀)·Φ + S
```

**特征值**：λ = −(γ − S·k/Φ₀)

**稳定性条件**（Hurwitz 判据）：
```
γ > S·k/Φ₀
```

**γ_max 的可能推导**：γ 太大时，线性项主导，系统收敛太快，可能压制自然振荡。具体需要非线性分析（如 Hopf 分岔点）确定。

**当前状态**：γ_max 是**数值占位**，没有严格理论推导。等分岔理论确认。

## δ 标定方法（补充适用条件）

### Φ 的分布假设（关键）

**问题**：percentile_95 和 mean+2σ 在正态分布下等价，但 Φ 的分布是什么？

根据 ECHO 机制（幂律效应、网络扩散）：
- Φ 的稳态分布可能是**重尾分布**（类似幂律）
- mean+2σ 在重尾分布下会严重低估 δ（因为均值不能代表尾部）
- percentile_95 更鲁棒，但不假设分布

### 三种方法的适用条件

| 方法 | 分布假设 | 适用场景 | 局限性 |
|------|---------|---------|--------|
| **percentile_95** | 无 | 小样本、分布未知 | 对样本量敏感 |
| **mean + 2σ** | 正态/近似正态 | 大样本、中心极限定理成立 | 重尾分布下失效 |
| **γ × Φ_max** | 模型驱动 | 有 PDE 稳态解析解 | γ 本身不准则跟着歪 |

### 建议

1. **优先用 percentile_95**（无分布假设）
2. **加 bootstrap 置信区间**作为兜底
3. **三种方法同时跑**，分歧大时以 percentile_95 为准
4. **未来**：从 PDE 稳态解解析推导 Φ 的分布
