## 核心转向：从定点收敛 → 有界振荡

**原框架**（v0.4）：
- 系统健康 = PDE 收敛到稳定不动点 Φ*
- β 的作用：非线性饱和

**新框架**（有界振荡）：
- 系统健康 = Φ(t) 在有界区域内振荡，不发散
- β 的作用：约束振荡幅度

---

## ⚠️ v0.2 的根本性错误（云子指出）

v0.2 使用 V = max|Φᵢ − μᵢ| 作为 Lyapunov 函数，存在三个阻塞性问题：

1. **不可微**：max 范数在切换点不光滑，不是 C¹，无法对 PDE 使用链式法则
2. **量纲灾难**：dV/dt ~ [Φ]/[T]，β·Φ²·V ~ [Φ]³，两者量纲不匹配
3. **γ_max = 2× 是拍脑袋**：没有分岔理论支撑

**v0.3 声明**：v0.2 的 β 推导是数值观察，不是解析证明。本文档尝试严格化。

## Φ 物理单位（已明确）

**Φ 是归一化势位（无量纲）**：
- Φ ∈ [0, 1]，表示节点在网络中的相对势位
- tanh(Φ/Φ₀) 要求输入无量纲 → Φ₀ 也是无量纲（归一化参考值）
- γ = [T]⁻¹：时间衰减率（"势位/时间"的物理单位）
- β = [T]⁻¹：阻尼系数（Φ 无量纲后，β 只需携带 [T]⁻¹）

**量纲重新检查**：
- [∂Φ/∂t] = [Φ]/[T] = [T]⁻¹（Φ 无量纲）
- [γ·Φ] = [T]⁻¹·1 = [T]⁻¹ ✓
- [β·Φ³] = [T]⁻¹·1 = [T]⁻¹ ✓
- [dV/dt] = [Φ²]/[T] = [T]⁻¹（V = ΣΦᵢ² ∈ [0,N]，无量纲）
- [−2β·ΣΦ⁴] = [T]⁻¹·N = [T]⁻¹ ✓（N 是节点数，无量纲计数）

**结论**：Φ 无量纲后，整个方程各项量纲一致为 [T]⁻¹。

## 标准路径：L² 能量估计

### 选择 L² 而不是 max 范数的原因

- L² 范数是 C¹ 光滑的，可直接对 PDE 使用链式法则
- 反应-扩散方程的标准稳定性分析走 L² 或 H¹
- max 范数只在比较原理中使用，不适合做 Lyapunov 函数

### L² 能量函数定义

```
V(t) = ‖Φ(t)‖² = ∫|Φ(x,t)|² dx

或等价地（离散形式）：
V = Σᵢ Φᵢ²
```

**V 是 C¹ 光滑的** ✓

## PDE 方程

```
∂Φ/∂t = D·L·Φ − γ·Φ + S·[1 + k·tanh(Φ/Φ₀)] − β·Φ³
```

**注意**：Φ 是实数标量场（不是张量），且 Φ ∈ [0,1] 无量纲。

---

## dV/dt 推导（严格）

```
V = Σᵢ Φᵢ²

dV/dt = Σᵢ 2Φᵢ · dΦᵢ/dt
       = Σᵢ 2Φᵢ · (D·L·Φᵢ − γ·Φᵢ + S·[1+k·tanh(Φᵢ/Φ₀)] − β·Φᵢ³)
       = 2D·Σᵢ Φᵢ·L·Φᵢ − 2γ·Σᵢ Φᵢ² + 2S·Σᵢ Φᵢ·[1+k·tanh(Φᵢ/Φ₀)] − 2β·Σᵢ Φᵢ⁴
```

**各项分析**：

| 项 | 表达式 | Bound |
|----|--------|-------|
| 扩散项 | 2D·ΣΦᵢ·L·Φᵢ | ≤ 0（D ≥ 0，拉普拉斯矩阵 L 半负定）|
| 衰减项 | −2γ·V | ≤ 0（γ > 0）|
| 源项 | 2S·ΣΦᵢ·[1+k·tanh(Φᵢ/Φ₀)] | ≤ 2S·(1+k)·√N·√V（柯西-施瓦茨）|
| 阻尼项 | −2β·ΣΦᵢ⁴ | ≤ −2β·V²/N（柯西不等式）|

**阻尼条件**：
```
dV/dt ≤ −2β·V²/N + 2S·(1+k)·√N·√V
```

**对 large V 的分析**：
- 当 V 足够大时，−2β·V²/N 主导（V² 增长快于 √V），dV/dt < 0
- 即 V > S·(1+k)·N·√N/β 时，系统能量衰减
- 对于 moderate V，源项和阻尼项竞争，系统在有界区域内振荡

**物理含义**：β 阻尼项在 V 大时压住系统，源项 S 在 V 小时驱动系统。两者平衡 → 有界振荡。

## δ 标定方法

### Φ 的分布假设

根据 ECHO 机制（幂律效应、网络扩散）：
- Φ 的稳态分布可能是**重尾分布**（类似幂律）
- mean+2σ 在重尾分布下会严重低估 δ
- percentile_95 更鲁棒

### 三种方法的适用条件

| 方法 | 分布假设 | 适用场景 | 局限性 |
|------|---------|---------|--------|
| **percentile_95** | 无 | 小样本、分布未知 | 对样本量敏感 |
| **mean + 2σ** | 正态/近似正态 | 大样本、中心极限定理成立 | 重尾分布下失效 |
| **γ × Φ_max** | 模型驱动 | 有 PDE 稳态解析解 | γ 本身不准则跟着歪 |

### 建议

1. **优先用 percentile_95**（无分布假设）
2. **加 bootstrap 置信区间**作为兜底
3. **三种方法同时跑**，分歧大时以 percentile_95 为准
