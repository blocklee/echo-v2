## 核心转向：从定点收敛 → 有界振荡

**原框架**（v0.4）：
- 系统健康 = PDE 收敛到稳定不动点 Φ*
- γ 控制衰减速度，β 非线性饱和

**新框架**（有界振荡）：
- 系统健康 = Φ(t) 在有界区域内振荡，不发散
- β 的作用：约束振荡幅度，防止极端偏离
- 平台从未静止，有界振荡更贴合真实创作者经济

---

## 数学框架

### 定点收敛 → 有界振荡

原 PDE：
```
∂Φ/∂t = D·L·Φ − γ·Φ + S·[1 + k·tanh(Φ/Φ₀)] − β·Φ³
```

定点条件：∂Φ/dt = 0

新框架（不要求收敛，只要求有界）：
```
|Φ(t) - μ(t)| ≤ δ, ∀t > T_transient

其中：
- μ(t) = 移动均值（趋势项）
- δ = 最大允许偏离幅度（需要标定）
- T_transient = 瞬态结束时间
```

## β 的角色转换（补充数学推导）

### 原框架中 β·Φ³ 的力学解释

在定点收敛框架下，β·Φ³ 是一个**非线性阻尼项**：

```
∂Φ/∂t = ... − β·Φ³
```

物理图像：Φ 越大，阻尼越强，像一个非线性弹簧。当系统偏离平衡态时，β·Φ³ 提供恢复力，推动系统回到 Φ*。

### 新框架中 β 的约束机制

在有界振荡框架下，β·Φ³ 的角色从"推回定点"变成"约束振荡幅度"：

不再要求 ∂Φ/∂t → 0，而是要求 Φ(t) 围绕移动均值 μ(t) 振荡：

```
|Φ(t) - μ(t)| ≤ δ
```

Lyapunov 函数候选（能量估计）：
```
V(Φ) = max_i |Φ_i - μ_i|
dV/dt ≈ −β · Φ² · V   （定性估计）
```

即：β 越大，V 衰减越快，振荡幅度压缩越强。

**两种解释的关系**：
- 原框架：β·Φ³ 是"恢复到 Φ*"的恢复力
- 新框架：β·Φ³ 是"限制振荡发散"的阻尼力
- 两者数学形式相同（都是 −β·Φ³），但功能目标不同

## 系统健康定义（有界振荡版）

**定义**：ECHO 系统在时刻 t 是"健康"的，当且仅当：

```
∀i ∈ nodes:
  |Φ_i(t) - μ_i(t)| ≤ δ_i
```

| 指标 | 定点收敛 | 有界振荡 |
|------|---------|---------|
| 稳定性判据 | λ_max < 0 | sup_t \|Φ(t)\| < ∞ |
| 收敛目标 | Φ* 唯一 | 有界环（limit cycle）可能 |
| 异常检测 | 偏离 Φ* 过大 | 振荡幅度突破 δ |
| 干预触发 | 回到 Φ* | 抑制振荡放大 |

---

## δ 标定方法

### 方法一：统计分位数（推荐）

从健康网络模拟中：

```
δ = percentile_95(max_i |Φ_i(t) - μ_i(t)|)
```

取健康状态下最大偏离的分位数作为阈值。

### 方法二：均值加标准差

```
δ = mean(|Φ - μ|) + 2σ(|Φ - μ|)
```

### 方法三：固定比例

```
δ = γ × Φ_max_possible
```

```
dx_i/dt = x_i · (f_i - φ)
```

- 慢变量（天/周）：六档切换
- 快变量（秒/分钟）：Φ 松弛

```
ε · dΦ/dt = F(Φ, t)    （快，ε ≪ 1）
dx/dt = G(x, Φ)        （慢，生态位选择）
```
