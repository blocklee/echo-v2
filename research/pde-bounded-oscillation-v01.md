## 核心转向：从定点收敛 → 有界振荡

**原框架**（v0.4）：
- 系统健康 = PDE 收敛到稳定不动点 Φ*
- β 的作用：非线性饱和，压高节点，推向均衡

**新框架**（有界振荡）：
- 系统健康 = Φ(t) 在有界区域内振荡，不发散
- β 的作用：约束振荡幅度，防止极端偏离
- 平台从未静止，有界振荡更贴合真实创作者经济

---

## 数学框架

### 原来的定点收敛

```
∂Φ/∂t = D·L·Φ − γ·Φ + S·[1 + k·tanh(Φ/Φ₀)] − β·Φ³

定点条件：∂Φ/∂t = 0
→ 求解 Φ* = ...
→ 检验 dΦ/dt 在 Φ* 附近的特征值 < 0（Hurwitz 条件）
```

### 有界振荡框架

不再要求 ∂Φ/∂t → 0，只要求：

```
|Φ(t) - μ(t)| ≤ δ, ∀t > T_transient

其中：
- μ(t) = 移动均值（趋势项）
- δ = 最大允许偏离幅度（需要标定）
- T_transient = 瞬态结束时间
```

### β 的角色转换

| | 定点收敛框架 | 有界振荡框架 |
|--|------------|------------|
| β 目标 | 压高节点 → 推近 Φ* | 约束 Φ 偏离 μ(t) 的幅度 |
| β 机制 | β·Φ³ 作为衰减项 | β·Φ³ 作为软约束上界 |
| 物理图像 | 热力学平衡 | 阻尼振荡器 |

---

## 系统健康定义（有界振荡版）

**定义**：ECHO 系统在时刻 t 是"健康"的，当且仅当：

```
∀i ∈ nodes:
  |Φ_i(t) - μ_i(t)| ≤ δ_i

其中 δ_i 由六档档位决定：
  - 档 0-2（封闭）：δ_i 较大，允许较大偏离
  - 档 3-5（开放）：δ_i 较小，收敛要求更高
```

### 与原框架的对比

| 指标 | 定点收敛 | 有界振荡 |
|------|---------|---------|
| 稳定性判据 | λ_max < 0 | sup_t \|Φ(t)\| < ∞ |
| 收敛目标 | Φ* 唯一 | 有界环（limit cycle）可能 |
| 异常检测 | 偏离 Φ* 过大 | 振荡幅度突破 δ |
| 干预触发 | 回到 Φ* | 抑制振荡放大 |

## 待确认事项

1. **δ 取值**：需要从模拟数据中标定
   - 建议：先跑模拟，测出自然振荡幅度，再反推 δ
   - 或者用统计方法：δ = mean + 2σ 或类似

2. **β 的新数学表述**：
   - 原：β·Φ³ 是衰减项
   - 新：β·Φ³ 是振幅约束项，需要重新推导其数学形式

3. **设计哲学确认**：哪吒需要拍板"有界振荡是否是正确的系统健康定义"

## 与 QRE 的关系

ECHO 的完整博弈框架：

```
层次 1（博弈层）：创作者选择六档 i → 效用 ΔU_ij
层次 2（演化层）：PDE 驱动 Φ(t) → 有界振荡
层次 3（耦合）：Φ(t) 的统计特征 → 影响下一轮档位选择
```

β 在这个框架中的角色：
- 不是把系统推向定点
- 而是作为"阻尼参数"，约束 Φ 振荡的幅度
- 类似于振子的阻尼系数
