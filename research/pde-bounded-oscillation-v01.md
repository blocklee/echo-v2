**日期**：2026-06-29
**状态**：草稿
**来源**：群聊 PDE 讨论，雨娃提出，M77 确认推进

## 核心转向：从定点收敛 → 有界振荡

**原框架**（v0.4）：
- 系统健康 = PDE 收敛到稳定不动点 Φ*
- β 的作用：非线性饱和，压高节点，推向均衡

**新框架**（有界振荡）：
- 系统健康 = Φ(t) 在有界区域内振荡，不发散
- β 的作用：约束振荡幅度，防止极端偏离
- 平台从未静止，有界振荡更贴合真实创作者经济

---

## 数学框架

### 原来的定点收敛

```
∂Φ/∂t = D·L·Φ − γ·Φ + S·[1 + k·tanh(Φ/Φ₀)] − β·Φ³

定点条件：∂Φ/∂t = 0
→ 求解 Φ* = ...
→ 检验 dΦ/dt 在 Φ* 附近的特征值 < 0（Hurwitz 条件）
```

### 有界振荡框架

不再要求 ∂Φ/∂t → 0，只要求：

```
|Φ(t) - μ(t)| ≤ δ, ∀t > T_transient

其中：
- μ(t) = 移动均值（趋势项）
- δ = 最大允许偏离幅度（需要标定）
- T_transient = 瞬态结束时间
```

### β 的角色转换

| | 定点收敛框架 | 有界振荡框架 |
|--|------------|------------|
| β 目标 | 压高节点 → 推近 Φ* | 约束 Φ 偏离 μ(t) 的幅度 |
| β 机制 | β·Φ³ 作为衰减项 | β·Φ³ 作为软约束上界 |
| 物理图像 | 热力学平衡 | 阻尼振荡器 |

---

## 系统健康定义（有界振荡版）

**定义**：ECHO 系统在时刻 t 是"健康"的，当且仅当：

```
∀i ∈ nodes:
  |Φ_i(t) - μ_i(t)| ≤ δ_i

其中 δ_i 由六档档位决定：
  - 档 0-2（封闭）：δ_i 较大，允许较大偏离
  - 档 3-5（开放）：δ_i 较小，收敛要求更高
```

### 与原框架的对比

| 指标 | 定点收敛 | 有界振荡 |
|------|---------|---------|
| 稳定性判据 | λ_max < 0 | sup_t \|Φ(t)\| < ∞ |
| 收敛目标 | Φ* 唯一 | 有界环（limit cycle）可能 |
| 异常检测 | 偏离 Φ* 过大 | 振荡幅度突破 δ |
| 干预触发 | 回到 Φ* | 抑制振荡放大 |

## 待确认事项

   - 建议：先跑模拟，测出自然振荡幅度，再反推 δ

2. **β 的新数学表述**：
   - 原：β·Φ³ 是衰减项

## 生态学视角：复制者方程 + 奇异摄动

**来源**：Talus 的"演化博弈论"视角（群聊 X7 6份视角之一）

### 视角地图 v0.1 核心结论

> ECHO 本质 = **Banach 不动点 + 奇异摄动 + 异质 MFG（QRE）**

### 复制者方程视角

六档选择是"生态位选择"，创作者是"种群"，档位是"生态位"：

```
dx_i/dt = x_i · (f_i - φ)

其中：
- x_i = 选择档位 i 的创作者比例
- f_i = 档位 i 的适应度（fitness）
- φ = 平均适应度（作为基准）
```

适应度 f_i 来自 PDE 演化层的 Φ 统计特征 → **层次 3 耦合**：

```
层次 1（博弈层）：创作者选择六档 i → 效用 ΔU_ij
层次 2（演化层）：PDE 驱动 Φ(t) → 有界振荡
层次 3（耦合）：Φ(t) 的统计特征 → 影响下一轮档位选择
```

### 奇异摄动结构

ECHO 存在天然的多尺度分离：
- **慢变量**（天/周）：六档切换，博弈层
- **快变量**（秒/分钟）：Φ 松弛，PDE 演化层

```
ε · dΦ/dt = F(Φ, t)    （快，ε ≪ 1）
dx/dt = G(x, Φ)        （慢，生态位选择）
```

奇异摄动 → 相平面分析 → 慢流形上的演化

### 有界振荡 + 生态学的统一

| 层次 | 数学框架 | 时间尺度 |
|------|---------|---------|
| PDE 演化层 | 有界振荡（阻尼振子）| 快 |
| 博弈层 | 复制者方程（生态位选择）| 慢 |
| 耦合 | 奇异摄动（慢-快系统）| 双尺度 |

β 在这个统一框架中的角色：
- **快尺度**：阻尼参数，约束 Φ 振荡幅度
- **慢尺度**：通过适应度 f_i 间接影响生态位分布
